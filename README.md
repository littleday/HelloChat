# HelloChat
A Chat Application support multi clients and multi rooms.

# Steps
1. Using the following command to enter the server
```
telnet IP PORT
```
2. The following commands are supported:
```
/help
/rooms
/join room_name
/ping user_name your words
/names
/leave
/quit
```
